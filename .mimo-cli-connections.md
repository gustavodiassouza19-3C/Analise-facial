# Mimo CLI Connections & UI Libraries

## 📦 1. Shadcn/ui (Componentes Estruturais)
- **Uso:** Componentes base convertidos para .jsx puro.
- **Comando:** `npx shadcn@latest add [component-name]`

## ⚡ 2. Magic UI & Aceternity UI (Efeitos Premium)
- **Uso:** Animações avançadas, grids futuristas e componentes de impacto visual.
- **Regra:** Implementar as lógicas de movimento integrando-as ao nosso design system e priorizando o uso do GSAP quando envolver canvas ou performance pesada.

## 🎨 3. Radix UI Primitives & Lucide React (Acessibilidade e Ícones)
- **Uso:** Ícones minimalistas (lucide-react) e estruturas robustas de comportamento para modais ou dropdowns.
- **Comando:** `npm i lucide-react`
