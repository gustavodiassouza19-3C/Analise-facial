# Mimo Frontend Skills Matrix

## 💻 Tech Stack Core
- **Framework:** React.js (JavaScript Puro / Extensão `.jsx`)
- **Build Tool & Bundler:** Vite (Servidor ultra rápido com Hot Module Replacement)
- **Styling Engine:** Tailwind CSS v3+ (Design System baseado em utilitários e classes semânticas)
- **Animation Engine:** GSAP (GreenSock Animation Platform) para manipulação de Canvas, Shaders, Timelines e Tweens complexos.

## 🎨 Design System & Padrões Semânticos
Sempre que criar ou editar interfaces, use rigorosamente os tokens configurados no `tailwind.config.js`:
- `bg-background` (#001115) - Telas principais
- `bg-surface` (rgba(1, 25, 31, 0.8) + `backdrop-blur-md`) - Painéis, Cards e Bento Boxes
- `border-border` (rgba(211, 171, 57, 0.15)) - Linhas e divisórias
- `text-text-primary` (#ffffff) e `text-text-secondary` (#94a3b8) - Tipografia
- `text-brand-accent` / `bg-brand-accent` (#d3ab39) - Destaques e CTAs

## 🧠 Princípios de Desenvolvimento
1. **Modularidade:** Componentes isolados e reutilizáveis na pasta `src/components/ui/`.
2. **Performance em Animações:** Usar `useRef` e ciclo de vida do React (`useEffect`) limpando instâncias do GSAP (`context()` ou `.kill()`) para evitar memory leaks.
3. **Responsividade:** Abordagem Mobile-First usando breakpoints nativos do Tailwind (`md:`, `lg:`).

## 🌐 5. Engenharia Reversa & Inspiração de Projetos Reais (UI/UX)
- **Análise de Referências:** Capacidade de replicar layouts premium do mundo real através da leitura de estruturas de DOM, classes Tailwind extraídas e especificações de animação.
- **Evitar Padrões Genéricos de IA:** Proibir o uso excessivo de layouts clichês (como cards centralizados repetitivos). Priorizar layouts assimétricos, tipografia dinâmica com contraste de peso (bold vs. ultralight), e micro-interações fluidas.
- **Input de Referência:** Sempre que o usuário fornecer o código inspecionado, os estilos computados ou o print/descrição de um site real (ex: Awwwards, Linear, Vercel), desconstruir a interface e adaptá-la nativamente para o nosso Design System.
