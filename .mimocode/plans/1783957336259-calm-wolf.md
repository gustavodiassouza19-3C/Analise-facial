## Nova Arquitetura

### 1. Normalizacao por DIP (Distancia Interpupilar)

```python
# Landmarks dos olhos
OLHO_ESQ_CENTRO = 33   # centro olho esquerdo
OLHO_DIR_CENTRO = 263  # centro olho direito

def calcular_dip(landmarks):
    """Distancia interpupilar como vetor de escala unitario."""
    esq = landmarks[OLHO_ESQ_CENTRO]
    dir = landmarks[OLHO_DIR_CENTRO]
    return math.sqrt((dir.x - esq.x)**2 + (dir.y - esq.y)**2)

# Todas as distancias sao: distancia_real / DIP
# Ex: face_width_normalizado = face_width_px / DIP
```

**Por que**: Elimina variacao de distancia da camera. Um rosto a 40cm e 80cm produzem os mesmos scores.

### 2. Validacao de Pose (Filtro de Entrada)

```python
def calcular_inclinacao(landmarks):
    """Calcula Roll, Pitch, Yaw a partir dos olhos e laterais do rosto."""
    # Roll: angulo dos olhos em relacao a horizontal
    olho_esq = landmarks[33]
    olho_dir = landmarks[263]
    roll = math.degrees(math.atan2(olho_dir.y - olho_esq.y, olho_dir.x - olho_esq.x))

    # Yaw: diferenca de escala entre laterais esquerda e direita
    lado_esq = landmarks[234]  # lateral esquerda do rosto
    lado_dir = landmarks[454]  # lateral direita do rosto
    orelha_esq = landmarks[127]
    orelha_dir = landmarks[356]
    largura_esq = abs(lado_esq.x - orelha_esq.x)
    largura_dir = abs(lado_dir.x - orelha_dir.x)
    yaw = math.degrees(math.atan2(largura_dir - largura_esq, largura_dir + largura_esq))

    return roll, yaw

# Se abs(roll) > 5 ou abs(yaw) > 5 -> HTTP 422 "Rosto inclinado"
```

### 3. Simetria Real por Reflexao

```python
# Pares de pontos correspondentes (esquerdo, direito)
PARES_SIMETRIA = [
    (33, 263),    # cantos internos dos olhos
    (133, 362),   # cantos externos dos olhos
    (61, 291),    # cantos da boca
    (234, 454),   # laterais do rosto
    (127, 356),   # limites das bochechas
]

def calcular_simetria(landmarks, dip):
    """Desvio absoluto medio entre lados esquerdo e direito, normalizado por DIP."""
    eixo_x = (landmarks[4].x + landmarks[152].x) / 2  # eixo central (nariz-queixo)

    desvios = []
    for idx_esq, idx_dir in PARES_SIMETRIA:
        dist_esq = abs(landmarks[idx_esq].x - eixo_x) / dip
        dist_dir = abs(landmarks[idx_dir].x - eixo_x) / dip
        desvios.append(abs(dist_esq - dist_dir))

    desvio_medio = sum(desvios) / len(desvios)
    # Converter para score 0-100 (desvio 0 = 100, desvio 0.5 = 0)
    score = max(0, min(100, 100 - desvio_medio * 200))
    return round(score, 1)
```

### 4. Tercos Anatômicos Corrigidos

```python
# Landmarks corretos
TRICHION = 10     # linha do cabelo
GLABELLA = 8      # entre as sobrancelhas
SUBNASALE = 2     # base do septo nasal
MENTON = 152      # queixo

def calcular_tercos(landmarks, dip):
    """Tercos faciais normalizados por DIP."""
    trichion_y = landmarks[TRICHION].y
    glabella_y = landmarks[GLABELLA].y
    subnasale_y = landmarks[SUBNASALE].y
    menton_y = landmarks[MENTON].y

    superior = (glabella_y - trichion_y) / dip
    medio = (subnasale_y - glabella_y) / dip
    inferior = (menton_y - subnasale_y) / dip
    total = superior + medio + inferior

    if total == 0:
        return {"superior": 0, "medio": 0, "inferior": 0}

    return {
        "superior": round((superior / total) * 100, 1),
        "medio": round((medio / total) * 100, 1),
        "inferior": round((inferior / total) * 100, 1),
    }
```

### 5. Angulo Nasolabial (fotos de perfil)

```python
SUBNASALE = 2
PRANASALE = 1     # ponta do nariz
LABIALE_SUPERIUS = 13  # labio superior

def calcular_angulo_nasolabial(landmarks):
    """Angulo entre septo nasal e labio superior (perfil)."""
    sub = landmarks[SUBNASALE]
    pra = landmarks[PRANASALE]
    lab = landmarks[LABIALE_SUPERIUS]

    v1 = (pra.x - sub.x, pra.y - sub.y)
    v2 = (lab.x - sub.x, lab.y - sub.y)

    dot = v1[0]*v2[0] + v1[1]*v2[1]
    mag1 = math.sqrt(v1[0]**2 + v1[1]**2)
    mag2 = math.sqrt(v2[0]**2 + v2[1]**2)

    if mag1 == 0 or mag2 == 0:
        return 0

    cos_angle = max(-1, min(1, dot / (mag1 * mag2)))
    return round(math.degrees(math.acos(cos_angle)), 1)
```

### 6. Plano E de Ricketts (fotos de perfil)

```python
def calcular_ricketts(landmarks, dip):
    """Distancia perpendicular dos labios a linha E (nariz-queixo), normalizada por DIP."""
    pra = landmarks[PRANASALE]      # ponta do nariz
    menton = landmarks[MENTON]      # queixo
    lab_sup = landmarks[LABIALE_SUPERIUS]
    lab_inf = landmarks[14]         # labio inferior

    # Linha E: de pra ate menton
    dx = menton.x - pra.x
    dy = menton.y - pra.y
    line_len = math.sqrt(dx**2 + dy**2)

    if line_len == 0:
        return {"superior": 0, "inferior": 0}

    def perp_dist(p):
        cross = dy * p.x - dx * p.y
        cross += menton.x * pra.y - menton.y * pra.x
        return cross / line_len

    return {
        "superior": round(perp_dist(lab_sup) / dip, 4),
        "inferior": round(perp_dist(lab_inf) / dip, 4),
    }
```

### 7. Score por Z-score (sem multiplicadores magicos)

```python
# Intervalos anatomicos saudaveis (media, desvio_padrao)
REFERENCIAS = {
    "terco_superior": {"ideal": 33.3, "sigma": 3.0},
    "terco_medio": {"ideal": 33.3, "sigma": 3.0},
    "terco_inferior": {"ideal": 33.3, "sigma": 3.0},
    "angulo_nasolabial": {"ideal": 100, "sigma": 10},  # graus
    "simetria": {"ideal": 0, "sigma": 0.1},  # desvio normalizado
}

def zscore_to_score(valor, ideal, sigma):
    """Converte Z-score para score 0-100."""
    z = abs(valor - ideal) / sigma if sigma > 0 else 0
    score = 100 * math.exp(-0.5 * z**2)
    return max(0, min(100, round(score, 1)))
```

## Alteracoes no useFaceMesh.js

### Novos estados
- `boundingBox` - { x, y, width, height, centerX, centerY }
- `distanceStatus` - "no-face" | "too-far" | "ideal" | "too-close"
- `isTrackingVisible` - boolean (controla visibilidade do overlay)

### Logica de bounding box
Calcular min/max X e Y a partir dos 468 landmarks do MediaPipe:
```
minX, minY = menor X/Y de todos os landmarks
maxX, maxY = maior X/Y de todos os landmarks
width = maxX - minX
height = maxY - minY
centerX = (minX + maxX) / 2
centerY = (minY + maxY) / 2
```

### Logica de distancia
- Face width / Video width:
  - < 35% = "too-far" (vermelho)
  - 35%-45% = "ideal" (verde)
  - > 45% = "too-close" (amarelo)

### Timer de captura automatica
Quando `distanceStatus === "ideal"`:
- Iniciar timer de 1.5s
- Se manter ideal, setar `isTrackingVisible = false` e disparar callback
- Se mudar de ideal, cancelar timer

- Cor baseada no status: verde (#22c55e), vermelho (#ef4444), amarelo (#f59e0b)
- Texto acima do retangulo: "Distancia ideal" / "Aproxime-se" / "Afaste-se"

### Novas funcoes expostas

# Analise Geral - MOGGED STUDIO (Analise Facial)

## Visao Geral do Projeto

**MOGGED STUDIO** e uma plataforma de biometria geométrica para analise facial, construida com:
- **Backend**: Python 3.12 + FastAPI + SQLAlchemy (async) + PostgreSQL + MediaPipe/OpenCV
- **Frontend**: React 18 + Vite + Tailwind CSS + MediaPipe FaceMesh (client-side)
- **Auth**: JWT (python-jose) com Bearer tokens
- **DB**: PostgreSQL via asyncpg + Alembic para migrations

### Fluxo Principal
1. Frontend detecta 9 pontos anatomicos faciais via MediaPipe FaceMesh (no browser)
2. Envia coordenadas (x,y) para `POST /api/v1/analysis/calculate-metrics`
3. Backend calcula metricas geometricas: tercos faciais, angulo nasolabial, Ricketts E-line
4. Frontend renderiza resultados em charts (Recharts) e badges
