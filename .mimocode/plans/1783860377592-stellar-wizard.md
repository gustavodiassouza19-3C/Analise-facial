# Plano: Alinhar variáveis globais com dourado #d3ab39

## Contexto
O Shadcn foi inicializado com cores padrão (amarelo/azul). Precisamos sobrescrever `--primary`, `--accent` e aliases para apontar ao dourado `#d3ab39` do design system.

## Arquivos a alterar

### 1. `frontend/src/index.css`
Adicionar variáveis CSS globais no `:root` (fora do bloco `.dark`):
```css
--primary: 43 75% 52%;        /* #d3ab39 em HSL */
--primary-foreground: 0 0% 4%;
--accent: 43 75% 52%;          /* #d3ab39 */
--accent-foreground: 0 0% 4%;
```

No bloco `.dark`, adicionar as mesmas variáveis (dark mode usa o mesmo dourado):
```css
--primary: 43 75% 52%;
--primary-foreground: 0 0% 4%;
--accent: 43 75% 52%;
--accent-foreground: 0 0% 4%;
```

### 2. `frontend/tailwind.config.js`
Adicionar no `colors` (top-level, junto a `background`, `surface`):
```js
primary: '#d3ab39',
accent: '#d3ab39',
```

## Verificação
- `nite build` deve compilar sem erros
- Classes `bg-primary`, `text-primary`, `bg-accent`, `text-accent` devem renderizar em dourado
